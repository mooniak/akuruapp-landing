# Akuru App / Flocks

Today 21st February 2026 is the World Mother Language Day and we are celebrating this day with a limited alpha launch of a range of new services and products for South Asian creative and knowledge work. These products will are called Akuru Apps named after the Sinhala word for letters and letterfroms. 

Akuru App is a desktop super app for Sri Lanka for individuals, bussinesses and orgnisations working with text, fonts and writing in Offcial Langauges of Sri Lanka. Alpha version will be limited in features, but implements the basic structure of the app for future expansion. 

Today it ships with a simple set of four toos; a font library, a canvas, a glossry and a text converter.

- Font libary is a font installer that makes it a breeze to install fonts. For now this only supports free fonts from Google Fonts library. Starting May 2025, this library will host a range of new fonts
- Canvas is for testing fonts without installing and downloading PNG and vector files to be used in other programmes. This will allow using new fonts at a fraction of the cost
- Glossary features a list of technical terms relased by the Official Langauges Departnemnt of Sri Lanka.
- Text converter is the most advanced ASCII to Unicode converter for Sinhala texts.
